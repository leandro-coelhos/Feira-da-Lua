\doxysubsection{Detailed Description}
\begin{DoxyVerb}Represents one of the possible fixture scopes in pytest.

Scopes are ordered from lower to higher, that is:

          ->>> higher ->>>

Function < Class < Module < Package < Session

          <<<- lower  <<<-
\end{DoxyVerb}
