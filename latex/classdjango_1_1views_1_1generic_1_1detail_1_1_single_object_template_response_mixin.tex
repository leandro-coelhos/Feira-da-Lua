django.\+views.\+generic.\+detail.\+Single\+Object\+Template\+Response\+Mixin.\+get\+\_\+template\+\_\+names (\begin{DoxyParamCaption}\item[{}]{self}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}Return a list of template names to be used for the request. May not be
called if render_to_response() is overridden. Return a list containing
``template_name``, if set on the value. Otherwise, return a list
containing:

* the contents of the ``template_name_field`` field on the
  object instance that the view is operating upon (if available)
* ``<app_label>/<model_name><template_name_suffix>.html``
\end{DoxyVerb}
