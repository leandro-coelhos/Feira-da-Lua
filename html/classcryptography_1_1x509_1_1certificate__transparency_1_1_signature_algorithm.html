Detailed Description

Signature algorithms that are valid for SCTs.

These are exactly the same as SignatureAlgorithm in RFC 5246 (TLS 1.2).

See: <https://datatracker.ietf.org/doc/html/rfc5246#section-7.4.1.4.1>
