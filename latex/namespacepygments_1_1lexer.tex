sublexers.

``insertions`` is a list of ``(index, itokens)`` pairs.
Each ``itokens`` iterable should be inserted at position
``index`` into the token stream given by the ``tokens``
argument.

The result is a combined token stream.

\doxysubsubsection{\texorpdfstring{using()}{using()}}
{\footnotesize\ttfamily \label{namespacepygments_1_1lexer_a14883da5137dfa2b06fa9bc2bbb1103e} 
pygments.\+lexer.\+using (\begin{DoxyParamCaption}\item[{}]{\+\_\+other}{, }\item[{\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}}]{kwargs}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}Callback that processes the match with a different lexer.

The keyword arguments are forwarded to the lexer, except `state` which
is handled separately.

`state` specifies the state that the new lexer will start in, and can
be an enumerable such as ('root', 'inline', 'string') or a simple
string which is assumed to be on top of the root state.

Note: For that to work, `_other` must not be an `ExtendedRegexLexer`.
\end{DoxyVerb}
